# CNN-Rapid-ABR-Peaks

## Installation

The codes were developed using Anaconda and Spyder on Windows. Other operating systems may work, but were not tested. The code can be run with either CPU-based PyTorch or CUDA-enabled GPU PyTorch.

The scripts used the following software environment:

* Python 3.12.9
* NumPy 2.4.6
* Matplotlib 3.10.1
* pandas 3.0.3
* SciPy 1.17.1
* PyTorch 2.6.0
* spyder-kernels 3.1.4
* CUDA 12.6
